Политика конфиденциальности

Дата последнего обновления: 29 октября 2025 г.

Ваша конфиденциальность важна для нас. Мы серьезно относимся к защите ваших персональных данных и соблюдаем требования Федерального закона № 152-ФЗ «О персональных данных».

1. Общие положения

Настоящая Политика конфиденциальности (далее — «Политика») определяет порядок обработки и защиты персональных данных пользователей (далее — «Пользователи») интернет-магазина минитракторов DONGFENG (далее — «Оператор», «Мы»).

Используя сайт и предоставляя свои персональные данные, Пользователь выражает согласие на обработку персональных данных в соответствии с настоящей Политикой.

2. Какие данные мы собираем

2.1. Персональные данные, предоставляемые Пользователем

При использовании сайта мы можем собирать следующую информацию:

  • Имя и фамилия — для идентификации при общении
  • Номер телефона — для связи по вопросам заказа
  • Адрес электронной почты — для отправки подтверждений и информации о заказе
  • Адрес доставки — для доставки товара
  • Данные о товаре — информация о выбранных минитракторах и комплектующих

2.2. Автоматически собираемые данные

При посещении сайта автоматически собираются следующие данные:

  • IP-адрес
  • Информация о браузере и устройстве
  • Данные о действиях на сайте (просмотренные страницы, время посещения)
  • Источник перехода на сайт (откуда пришли)
  • Файлы cookie (см. раздел 7)

3. Цели обработки персональных данных

Мы используем ваши персональные данные для следующих целей:

  • Обработка и выполнение заказов
  • Связь с клиентами по вопросам заказов, доставки и гарантийного обслуживания
  • Улучшение качества обслуживания и работы сайта
  • Информирование о новых товарах, акциях и специальных предложениях (с вашего согласия)
  • Анализ поведения пользователей для оптимизации сайта
  • Предотвращение мошенничества и обеспечение безопасности
  • Соблюдение законодательных требований

4. Правовые основания обработки данных

Обработка персональных данных осуществляется на основании:

  • Согласия Пользователя на обработку персональных данных
  • Договора купли-продажи (выполнение обязательств по доставке и гарантии)
  • Федерального закона № 152-ФЗ «О персональных данных»
  • Законодательства РФ о защите прав потребителей

5. Передача данных третьим лицам

5.1. Когда мы передаем данные

Ваши персональные данные могут быть переданы третьим лицам только в следующих случаях:

  • Транспортные компании — для организации доставки (имя, телефон, адрес)
  • Платежные системы — для обработки платежей (без хранения данных банковских карт)
  • Сервисы аналитики — Yandex.Metrika (обезличенные данные для аналитики)
  • Государственные органы — по официальным запросам в рамках законодательства

5.2. Защита данных при передаче

Все партнеры, получающие доступ к вашим данным, обязаны соблюдать конфиденциальность и использовать данные только для выполнения своих обязанностей.

6. Хранение и защита данных

6.1. Сроки хранения

  • Данные заказов хранятся в течение 3 лет для целей бухгалтерского учета и гарантийного обслуживания
  • Контактные данные (телефон, email) хранятся до момента отзыва согласия на обработку
  • История посещений и аналитика — до 25 месяцев (Yandex.Metrika)

6.2. Меры защиты

Для защиты ваших персональных данных мы применяем следующие меры:

  • Шифрование данных при передаче (SSL/TLS протокол — HTTPS)
  • Ограничение доступа к базам данных только уполномоченным сотрудникам
  • Резервное копирование данных для предотвращения потери
  • Регулярное обновление систем безопасности
  • Соблюдение требований стандартов безопасности

7. Файлы cookie

7.1. Что такое cookie

Cookie — это небольшие текстовые файлы, которые сохраняются на вашем устройстве при посещении сайта. Они помогают запоминать ваши предпочтения и улучшают работу сайта.

7.2. Какие cookie мы используем

  • Необходимые cookie — обеспечивают работу корзины, избранного и сравнения
  • Аналитические cookie — Yandex.Metrika для анализа трафика и поведения пользователей
  • Функциональные cookie — сохранение настроек фильтров и предпочтений

7.3. Управление cookie

Вы можете управлять cookie в настройках браузера: разрешить, заблокировать или удалить их. Обратите внимание, что отключение cookie может ограничить функциональность сайта (например, перестанет работать корзина).

8. Ваши права

В соответствии с законодательством РФ вы имеете следующие права:

  • Право на доступ — запросить копию ваших персональных данных, которые мы храним
  • Право на исправление — запросить исправление неточных или неполных данных
  • Право на удаление — запросить удаление ваших персональных данных («право на забвение»)
  • Право на ограничение обработки — ограничить способы использования ваших данных
  • Право на отзыв согласия — отозвать согласие на обработку данных в любой момент
  • Право на возражение — возразить против использования данных в маркетинговых целях
  • Право на жалобу — подать жалобу в Роскомнадзор

8.1. Как реализовать свои права

Для реализации своих прав направьте письменный запрос по телефону +7 (969) 999-56-68. Мы обработаем ваш запрос в течение 30 дней с момента получения.

9. Безопасность детей

Наш сайт не предназначен для лиц младше 18 лет. Мы сознательно не собираем персональные данные детей. Если вы узнали, что ребенок предоставил нам свои данные без согласия родителей, пожалуйста, свяжитесь с нами для удаления этой информации.

10. Изменения в политике конфиденциальности

Мы можем периодически обновлять настоящую Политику. При внесении существенных изменений мы разместим уведомление на сайте. Дата последнего обновления указана в начале документа.

Рекомендуем периодически проверять эту страницу для ознакомления с актуальной версией Политики.

11. Международная передача данных

Ваши персональные данные обрабатываются на территории Российской Федерации. Мы не передаем данные в страны, не обеспечивающие адекватный уровень защиты персональных данных.

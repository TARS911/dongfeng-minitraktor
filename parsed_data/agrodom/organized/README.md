## Использование

Каждый JSON файл содержит массив товаров с полями:
- `name` - название
- `price` - цена
- `image_url` - изображение
- `link` - ссылка на источник
- `sku` - артикул
- **`brand`** - бренд/производитель
- **`category`** - категория
- **`subcategory`** - подкатегория (если определена)

### 04_dongfeng_parts.sql - ГЛАВНЫЙ ФАЙЛ
**Размер:** 40 KB | **Строк:** 315 | **Товаров:** 207

Полный SQL файл со всеми 207 товарами категории "DongFeng, MasterYard (от 24 л.с)"

**Содержит:**
- CREATE TABLE `dongfeng_parts` с полной схемой (10 полей)
- 207 INSERT INTO statements, разделены на 12 групп по страницам
- Подробные комментарии с статистикой в начале и конце файла
- Готов к прямому импорту в MySQL/MariaDB

**Структура товаров:**
```
✓ 185 запчастей (Запчасти DongFeng/MasterYard) - страницы 1-10
✓ 22 стекла (Стекла DongFeng 244) - страницы 1-2
────────────────────────────────────────────────────
= 207 товаров всего
```

**Схема таблицы:**
- `id` - Первичный ключ, автоинкремент
- `product_name` - Название товара (VARCHAR 500)
- `price_rubles` - Цена в рублях (DECIMAL 10,2)
- `sku_code` - Артикул товара (VARCHAR 50)
- `in_stock` - Наличие ('yes', 'no', 'unknown')
- `category` - Основная категория
- `subcategory` - Подкатегория
- `product_url` - URL товара (TEXT)
- `date_added` - Дата добавления (TIMESTAMP)
- `UNIQUE KEY` на sku_code

### 3. Проверка после импорта

#### Общая информация
```sql
SELECT COUNT(*) as total_products FROM dongfeng_parts;
-- Результат: 207
```

#### По категориям
```sql
SELECT category, COUNT(*) as count 
FROM dongfeng_parts 
GROUP BY category 
ORDER BY count DESC;
```

#### По наличию
```sql
SELECT in_stock, COUNT(*) as count 
FROM dongfeng_parts 
GROUP BY in_stock;
```

#### Статистика по ценам
```sql
SELECT 
    MIN(price_rubles) as min_price,
    MAX(price_rubles) as max_price,
    AVG(price_rubles) as avg_price,
    COUNT(*) as total_items
FROM dongfeng_parts;
```

#### Товары с артикулами
```sql
SELECT COUNT(*) as with_sku FROM dongfeng_parts WHERE sku_code IS NOT NULL;
```
